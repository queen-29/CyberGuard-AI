# CyberGuard-AI
A Streamlit-based cybersecurity dashboard for risk assessment, incident management, policy analysis, and security awareness.

## Overview

CyberGuard AI is a cybersecurity dashboard designed to support both Governance, Risk & Compliance (GRC) and Information Security operations.

The platform provides tools for:

- Vendor Risk Assessment
- Incident Reporting
- Security Awareness Training
- Policy Analysis

This project was built using Python and Streamlit.

## Features

### Vendor Risk Assessment
Evaluate third-party vendors based on cybersecurity controls and generate risk ratings.

### Incident Reporting
Report and track cybersecurity incidents with severity classification.

### Security Awareness Quiz
Test cybersecurity knowledge through interactive quizzes.

### Policy Analyzer
Review security policies and identify potential gaps.
